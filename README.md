# Panaversity Labs

Hands-on labs for [Panaversity](https://panaversity.org) courses. Each course has its own directory with exercises students complete alongside live classes.

## Courses

| Course | Description | Labs |
|--------|-------------|------|
| [AI-101](AI-101/) | AI-Driven Development, Claude Code, and OpenClaw Fundamentals | 1 lab |
| [crash-course](crash-course/) | Downloadable projects for the Getting Started crash courses in _The AI Agent Factory_ | 5 projects |

## How to Use

1. Find your course directory
2. Pick the lab that matches your current module
3. Follow the exercises in order
4. Write down your observations after each exercise
