# Crash Course Projects

Downloadable projects for the [Getting Started crash courses](https://agentfactory.panaversity.org/docs/) in _The AI Agent Factory_.

Each project is small on purpose. It exists so you can **run one idea** and watch it work, rather than only read about it.

## Courses

| Course | Projects |
| ------ | -------- |
| [Loop Engineering](loop-eng/) | 1 project |

## How to use these

1. Download the project folder for the concept you are reading about.
2. Open it in your agent (`cd` into it first — the folder carries its own setup).
3. Follow its `README.md`.
4. Break it, change it, keep it. It is yours.
